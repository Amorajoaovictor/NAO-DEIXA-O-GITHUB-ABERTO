## 🎭 Pegadinha para quem deixa o GitHub aberto na faculdade!

Este repositório contém scripts em Python para zoar (de forma educativa) quem esquece o GitHub logado em computadores públicos da faculdade.

### 📁 Arquivos disponíveis:

- **`prank_github.py`** - Versão completa com efeitos especiais, contador e muita zoação! 🎪
- **`quick_prank.py`** - Versão rápida para uma zoação express! ⚡

### 🚀 Como usar:

1. **Versão completa (recomendada para máxima zoação):**
   ```bash
   python3 prank_github.py
   ```

2. **Versão rápida (para zoação express):**
   ```bash
   python3 quick_prank.py
   ```

### 🎯 Objetivo educativo:

- 🔒 Ensinar a importância de fazer logout em computadores públicos
- 🧠 Consciientizar sobre segurança digital na faculdade
- 😅 Divertir de forma saudável (sem causar danos)

### ⚠️ Avisos importantes:

- ✅ Scripts totalmente inofensivos (não alteram arquivos nem sistema)
- ✅ Apenas exibem mensagens engraçadas no terminal
- ✅ Objetivo educativo sobre segurança digital
- ❌ Não utilizar para intimidação ou bullying

### 🎪 Dicas de uso:

1. Execute quando encontrar um GitHub aberto e abandonado
